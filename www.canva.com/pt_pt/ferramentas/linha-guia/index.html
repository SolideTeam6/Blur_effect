Deixe as suas fotos e layouts perfeitos usando linhas guia de design

Comece a criar com o Canva

As linhas guia são uma excelente forma de criar o layout das suas imagens – quer seja um painel semântico para dar o pontapé de saída na sua criatividade ou imagens incríveis para o seu perfil nas redes sociais.

Desde capas icónicas como A Hard Day’s Night dos Beatles às grandes obras de arte de Piet Mondrian, os designs com base em linhas guia têm sempre um apelo especial para o nosso sentido de estrutura e forma. Também pode experimentar com linhas guia de design clicando no painel de “Layout” e selecionando uma das várias opções que oferecemos. Basta arrastar e largar qualquer elemento nas molduras disponíveis e pronto!

Use as linhas guia como base para o seu projeto. Desde designs simples com três molduras a combinações complexas, perfeitas para painéis semânticos, as linhas guia de design oferecem oportunidades para brincar e inovar com os seus desenhos. As possibilidades são infinitas!

Escolha uma linha guia para dar asas à sua imaginação

Um método que é tiro e queda para impulsionar a sua criatividade é colocar alguns limites para que o seu pensamento lateral possa agir, encontrando novas maneiras de lidar com problemas de design. Começar o design com uma linha guia é uma excelente forma de planear e preparar as suas imagens. Da próxima vez que tiver um bloqueio criativo para uma montagem de fotos, navegue pela nossa seleção de linhas guia e comece o seu brainstorming.

Selecione o layout de linha guia para recortes artísticos

Com linhas guia de design, o trabalho duro já está pronto. Arraste qualquer foto para uma caixa de linha guia e nós encaixamos. Mas a diversão começa quando clica em “Cortar” para redimensionar, recortar e posicionar a imagem. Os cortes podem criar tensão e foco ou dar uma sensação de amplitude. Pode experimentar diferentes formas de cortar uma foto num mesmo design para dar uma ideia de plenitude e evolução.

Acrescente destaques e contraste de cores nas suas linhas guia

As linhas guia de design são extremamente versáteis. Por exemplo, pode compor as linhas guia com mais do que simples fotos. Pode colorir as caixas com uma cor sólida selecionando a caixa e escolhendo uma cor a partir da paleta ou da roda de cores. Crie linhas guia coloridas para dar equilíbrio e textura ou use uma sombra para unificar o seu design.

Adicione texto para criar narrativas e contextos com a sua linha guia

O Canva permite criar camadas de elementos, oferecendo diversas oportunidades para deixar as suas linhas guia de design ainda melhores. Adicione texto em molduras específicas ou entre as molduras. Pode usar um dos nossos textos modelo com combinações perfeitas de tipos de letra ou criar o seu próprio arranjo com a nossa versátil ferramenta de texto.

Use linhas guia para contar histórias

Tal como as páginas de banda desenhada, as linhas guia podem ser utilizadas para dar ideia de tempo e movimento. Mas não pense que as linhas guia servem apenas para contar desventuras de heróis em lycra! As linhas guia são perfeitas para instruções, diagramas e até mesmo para criar banda desenhada fazendo o upload das suas próprias fotos e adicionando balões de fala. Você está do lado da força!
